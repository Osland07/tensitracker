# Kuesioner System Usability Scale (SUS) - TensiTrack

Kuesioner ini bertujuan untuk mengukur tingkat kegunaan (*usability*) dari sistem **TensiTrack**.
Responden dimohon untuk memberikan penilaian pada setiap pernyataan dengan skala 1 sampai 5.

**Skala Penilaian:**
1. Sangat Tidak Setuju
2. Tidak Setuju
3. Netral
4. Setuju
5. Sangat Setuju

---

### Cara Menghitung Skor SUS:

1.  Untuk item pertanyaan **Ganjil** (1, 3, 5, 7, 9): Skor item = **(Nilai Jawaban - 1)**
2.  Untuk item pertanyaan **Genap** (2, 4, 6, 8, 10): Skor item = **(5 - Nilai Jawaban)**
3.  Jumlahkan semua skor item (Total Skor 0-40).
4.  Kalikan total skor dengan **2.5**.
5.  Hasil akhir adalah skor SUS (0-100).

**Interpretasi Skor:**
*   **> 80.3**: Grade A (Excellent)
*   **68 - 80.3**: Grade B (Good)
*   **68**: Rata-rata (Average)
*   **51 - 68**: Grade C (Okay)
*   **< 51**: Grade F (Poor)
